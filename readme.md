产品名称：AI 灵动提词器 (AI Smart Prompter)
1. 核心产品愿景
通过实时语音监测与大模型语义匹配，实现提词内容与用户语速的同步，让用户在录制视频时无需担心忘词或语速失控。

2. 核心功能模块

A. 基础提词功能
自定义设置：字体大小、滚动速度（备用）、背景透明度、高亮颜色。
悬浮窗模式：支持在其他 App（如抖音/相机）上方开启悬浮窗提词。

B. 智能跟随模式 (AI Voice-Follow Mode) —— 核心亮点
语音流监测：调用手机麦克风，将用户的语音实时转化为文字流（ASR）。
语义对齐算法：利用大模型（或简单的模糊匹配算法）对比“录入语音”与“提词文案”。
动态滚动：根据识别到的当前句子位置，自动将文案滚动至屏幕视觉中心，并高亮当前朗读到的词句。
停顿感应：用户停顿时，提词器自动静止；用户跳读时，提词器自动定位到新位置。
